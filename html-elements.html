HTML Elements Reference Guide

HTML = HyperText Markup Language

It's the standard language used to create and structure web pages. Think of it as the backbone of any website â€” it defines the content (like text, images, links, videos, forms).

Headings

HTML provides six levels of headings, from <h1> to <h6>, with <h1> being the most important and largest, and <h6> the least important and smallest.

<h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
Output

Heading Level 1

Heading Level 2

Heading Level 3

Paragraph

A paragraph is defined with the <p> tag. It is used to represent a block of text.

<p>This is a paragraph element</p>
                
<p>Another paragraph here.</p>
Output

This is a paragraph element

Another paragraph here.

Text Formatting

Text formatting elements are used to style and emphasize text. The most common formatting elements are <strong> for bold text and <em> for italic text.

<strong>Bold text</strong>
<em>Italic text</em>
Output

Bold text
italic text

Lists

HTML supports two main types of lists: unordered lists (<ul>) and ordered lists (<ol>). List items are defined with the <li> tag.

<ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>
Output
  • List item 1
  • List item 2
  • List item 3

Ordered List

An ordered list is used when the order of items matters. It is created using the <ol> tag, with each item defined by the <li> tag.

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>
Output
  1. First item
  2. Second item
  3. Third item

Links and Images

Links are created using the <a> tag, which stands for "anchor". The href attribute specifies the URL of the page the link goes to. 

<a href="https://example.com">This is a hyperlink</a>
Output This is a hyperlink

Image Example

Images are embedded in HTML using the <img> tag. The src attribute specifies the path to the image, while the alt attribute provides alternative text for the image.

<img src="sample-image.jpeg" width="150" height="150" alt="Description of image">
Output

Table Example

Tables are created using the <table> tag. Inside a table, you can define table headers with <th>, table rows with <tr>, and table data cells with <td>.

<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Row 1, Cell 1</td>
            <td>Row 1, Cell 2</td>
        </tr>
    </tbody>
</table>
Output
Header 1 Header 2
Row 1, Cell 1 Row 1, Cell 2

Form Elements

Forms are used to collect user input. Common form elements include text inputs, password fields, and buttons. The <form> tag wraps all form elements.

<form>
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">

    <label for="password">Password:</label>
    <input type="password" id="password" name="password">

    <button type="submit">Submit</button>
</form>
Output




Semantic Elements

Semantic elements provide meaning to the web content. Examples include <header>, <footer>, <article>, and <section>.

<header>
    <nav>
        <ul>
            <li><a href="#home">Home</a></li>
            <li><a href="#about">About</a></li>
        </ul>
    </nav>
</header>
Output
  • Home
  • About

Other Elements

Other common HTML elements include <blockquote> for quotations and <pre> for preformatted text.

<blockquote>This is a blockquote element</blockquote>
Output
This is a blockquote element

Preformatted Text

The <pre> tag is used for preformatted text. Text within this tag preserves both spaces and line breaks.

<pre>
This is preformatted text
It preserves both spaces and line breaks
</pre>
Output 
This is preformatted text
It preserves both spaces and line breaks
